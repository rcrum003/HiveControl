Quick Installation

Copy and paste this command into your Raspberry Pi terminal:

curl -sSL https://raw.githubusercontent.com/rcrum003/HiveControl/master/easy-install.sh | sudo bash

What This Does

  • Downloads and runs the installer automatically
  • Installs all dependencies (Apache, PHP, SQLite)
  • Compiles sensor software
  • Sets up the web interface
  • Takes 30-60 minutes to complete

Manual Installation

For users who want to review the code before running:

1 Download the installer
curl -O https://raw.githubusercontent.com/rcrum003/HiveControl/master/install.sh
2 Review the script (optional)
less install.sh
Press 'q' to quit viewing
3 Make it executable
chmod +x install.sh
4 Run the installer
sudo ./install.sh
Options: -b (Bee Counter) | -x (Remote Desktop) | -k (Touchscreen Keyboard) | -d (Debug)
